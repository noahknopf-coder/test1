# Asia Trip Journal

A static multi-page website about an Asia trip. It is dependency-free and can be hosted for free on GitHub Pages.

## Pages

- `index.html` — overview and trip framing.
- `itinerary.html` — week-by-week timeline.
- `countries.html` — country summaries.
- `gallery.html` — Asia-only photo gallery.
# Apple Picking Joy

A static website about the pleasures of apple picking that costs **$0** to run on free static hosting.

## Live link format (after enabling GitHub Pages)

Once this repo is pushed to GitHub and Pages is enabled, your link will be:

- `https://<your-github-username>.github.io/<repo-name>/`

If the repository is named `<your-github-username>.github.io`, the link becomes:

- `https://<your-github-username>.github.io/`

## Why it costs no money

- Single `index.html` file
- No backend
- No database
- No paid APIs or third-party services

## Local preview

```bash
python -m http.server 4173 --bind 127.0.0.1
```

Then open `http://127.0.0.1:4173/`.
